# Дашборд продаж автодилера

## Описание
Интерактивный дашборд для отслеживания продаж автодилера с возможностью:
- Загрузки данных из JSON
- Автоматической агрегации по месяцам
- Прогнозирования выполнения планов
- Визуализации через графики и радар-диаграммы
- Системы авторизации с ролями (админ/наблюдатель)

## Технологии
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Backend: Node.js, Express
- База данных: PostgreSQL (Neon)
- Графики: Chart.js
- Деплой: Vercel

## Установка и запуск

### Локально
```bash
git clone <repository-url>
cd dashboard-auto
npm install
npm start
